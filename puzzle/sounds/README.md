# 音效文件目录

将以下 MP3 文件放入此目录，游戏会自动加载播放。文件缺失时游戏正常运行，只是没有声音。

| 文件名 | 用途 | 建议 |
|--------|------|------|
| `pickup.mp3` | 拾起拼图切片 | 短促轻快的音效（< 0.2 秒） |
| `drop.mp3` | 放下拼图切片 | 低频短音效（< 0.2 秒） |
| `snap.mp3` | 切片拼合成功 | 清脆的"叮"一声（< 0.3 秒） |
| `complete.mp3` | 完成整幅拼图 | 上行琶音 / 胜利音效（< 2 秒） |
| `shuffle.mp3` | 打乱切片 | 快速随机音效（< 0.5 秒） |
| `button.mp3` | 按钮点击 | 短促点击音（< 0.1 秒） |
| `bgm.mp3` | 背景音乐 | 循环播放的柔和背景音乐 |

## 替换方法

直接用同名的 MP3 文件覆盖即可，无需修改代码。
